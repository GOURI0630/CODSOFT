# 🧮 Calculator

This is a simple Python-based calculator project that performs basic arithmetic operations.

## Features
- Addition ➕
- Subtraction ➖
- Multiplication ✖️
- Division ➗ (with zero division error handling)

## How to Run
```bash
python calculator.py
##example
Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
Result: 50
