# 📝 To-Do List (Python)

## 📌 Project Overview
The **To-Do List** is a simple Python project that helps users manage their daily tasks.  
It allows adding, viewing, updating, and deleting tasks in an organized manner.  

This project was created as part of my **CodSoft Python Internship (Batch B52 - September 2025).**

---

## ⚙️ Features
- ➕ Add new tasks  
- 📋 View all tasks  
- ✏️ Update/Edit tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  

---

## 🛠️ Technologies Used
- Python (Core programming)  
- Jupyter Notebook (`.ipynb`)  

---

## ▶️ How to Run
1. Download or clone this repository.  
2. Open the file **TO-DO LIST.ipynb** in Jupyter Notebook / JupyterLab.  
3. Run the code cells step by step.  
4. Interact with the program using the menu options.
